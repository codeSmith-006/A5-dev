Question-3: What is event delegation in the DOM?

Suppose I have a list and inside the list I have some list items. Now i want to add onclick for each items, i can set onclick on each items or instead i can set onclick to their parent and all the items get the same functionality, that is call event delegation

Question-4: What is event bubbling in the DOM?

Suppose i set a click function in the body with print body is clicked then a list with print list is clicked then list items with print items is clicked. When I click in the items -> it will the compiler will find the last click target item that was clicked, in that case last item is items so it will print items then compiler will go reverse and find if there is any click function or not. So, it will go to items parent that is list and print list is clicked then it will go it's parent which is body and it will print body is clicked like bubbling. This concept is called event bubbling
